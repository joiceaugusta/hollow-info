# Hollow Info - Guia de Referência de Hollow Knight

Este projeto é uma plataforma informativa dedicada ao universo de Hollow Knight. O objetivo principal é oferecer um guia centralizado sobre as mecânicas fundamentais do jogo, auxiliando jogadores na compreensão de sistemas de combate, progressão e habilidades.

## Demonstração Online
O projeto está publicado e pode ser acessado através do link abaixo:
https://joiceaugusta.github.io/hollow-info/

## Proposta do Projeto
A página foi desenvolvida para servir como um compêndio de consulta rápida. O conteúdo está organizado em seções que abordam os pilares da jogabilidade:

* **Gestão de Magia e Alma:** Explicação detalhada sobre o sistema de mana, como obter as magias principais (Espírito Vingativo, Mergulho Desolador e Espectros Uivantes) e como realizar seus respectivos aprimoramentos.
* **Sistema de Amuletos:** Informações sobre o funcionamento dos encaixes e o estado de sobrecarga, essencial para a estratégia de combate.
* **Artes do Ferrão:** Um guia sobre as técnicas ensinadas pelos mestres Oro, Mato e Sheo, incluindo links externos para tutoriais em vídeo.

## Desenvolvimento Técnico
O projeto foi construído utilizando tecnologias web fundamentais, focando em uma interface imersiva e responsiva:

* **HTML5:** Estruturação semântica para garantir acessibilidade e melhor indexação.
* **CSS3:** Uso de variáveis customizadas para manter a consistência da paleta de cores (Abismo e Alma), além de animações de pulsação, efeitos de vidro (backdrop-filter) e estilização de imagens com filtros de brilho e escala de cinza.
* **JavaScript:** Implementação de funções para rolagem suave (smooth scroll) entre as seções, melhorando a experiência de navegação do usuário.
* **Tipografia:** Integração das fontes Cinzel e Lora via Google Fonts para reforçar a estética de fantasia sombria do título.

## Organização de Arquivos
Para manter o projeto escalável e organizado, a estrutura de diretórios segue este padrão:

* `index.html`: Documento principal da aplicação.
* `hk info/estilos/`: Arquivos CSS de estilização.
* `hk info/images/`: Repositório de capturas de tela e ativos visuais.
* `hk info/javascript/`: Scripts de comportamento e interatividade.

## Como Executar o Projeto Localmente
Caso deseje rodar o projeto em sua máquina de desenvolvimento, siga os passos abaixo:

1. Clone este repositório utilizando o terminal:
`git clone https://github.com/joiceaugusta/hollow-info.git`

2. Acesse a pasta do projeto:
`cd hollow-info`

3. Abra o arquivo `index.html` em seu navegador de preferência.
